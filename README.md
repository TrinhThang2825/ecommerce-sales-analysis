# Dự án Phân tích Dữ liệu Bán hàng (E-commerce Analysis)

## 📌 Giới thiệu
Dự án tập trung vào việc làm sạch và phân tích dữ liệu bán hàng để tìm ra các mặt hàng mang lại lợi nhuận cao nhất và xu hướng mua sắm của khách hàng.

## 🛠 Công cụ sử dụng
- **Ngôn ngữ:** Python (Pandas, Matplotlib, Seaborn)
- **Môi trường:** Google Colab / Jupyter Notebook
- **Nguồn dữ liệu:** Kaggle

## 📂 Cấu trúc dự án
- `/data`: Chứa file dữ liệu thô và dữ liệu sạch.
- `/notebooks`: Chứa các file code Python (.ipynb).
- `/reports`: Hình ảnh biểu đồ và nhận xét kết quả.

## 📈 Tiến độ thực hiện
- [x] Tuần 1: Khởi tạo Repo và tìm kiếm Dataset.
- [ ] Tuần 2-3: Xử lý dữ liệu (Lọc trùng, xử lý giá trị trống).
- [ ] Tuần 4-6: Vẽ biểu đồ xu hướng và phân tích.
- [ ] Tuần 7-8: Tổng kết Insights và báo cáo cuối kỳ.
